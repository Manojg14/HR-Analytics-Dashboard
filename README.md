# 📊 HR Analytics Dashboard | Power BI

An interactive HR Analytics Dashboard developed using **Microsoft Power BI** to help HR teams monitor workforce metrics, employee attrition, demographics, salary distribution, job satisfaction, and employee performance.

## 📌 Project Overview

Organizations generate large amounts of employee data every day. This dashboard transforms raw HR data into meaningful insights that help HR managers make data-driven decisions.

The dashboard focuses on:

- Employee Attrition Analysis
- Workforce Demographics
- Salary Analysis
- Department Performance
- Job Satisfaction
- Employee Experience
- HR KPIs

## 🎯 Project Objectives

- Analyze employee attrition
- Identify high-risk departments
- Monitor workforce demographics
- Compare salary across job roles
- Evaluate employee satisfaction
- Support HR decision-making
- Build an interactive HR reporting dashboard
